HTML and CSS

What is HTML and CSS?

HTML and CSS are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS provides the visual and aural layout, for a variety of devices.

What is HTML?

HTML, the HyperText Markup Language, is the language for describing the structure of Web pages.

What is CSS?

CSS, the Cascading Style Sheets, is the language for describing the presentation of Web pages, including colors, layout, and fonts. It allows one to adapt the presentation to different types of devices. The separation of HTML from CSS makes it easier to maintain sites, share style sheets across pages, and tailor pages to different environments.

Basic HTML tags

h1 - h6 tags

The h1 to h6 tags are used to define HTML headings. h1 defines the most important heading. h6 defines the least important heading.

p tag

The p HTML element represents a paragraph.

br

The br HTML element produces a line break in text.

ul, ol, li

The li HTML element is used to represent an item in a list. It must be contained in a parent element: an ordered list (ol), an unordered list (ul).

strong, em

The strong tag is used to define text with strong importance. The em tag is used to define emphasized text.

a

The a tag defines a hyperlink, which is used to link from one page to another.

img

The img tag is used to embed an image in an HTML page.

table, thead, tbody, tr, th, td

The table tag defines an HTML table. The thead tag is used to group header content in an HTML table. The tbody HTML element encapsulates a set of table rowsth defines a header cell in a table. tr defines a row in a table. td defines a cell in a table.

div

The div tag defines a division or a section in an HTML document.

CSS properties

color

The color CSS property sets the color value of an element's text.

background-color

The background-color property sets the background color of an element.

font-family

The font-family property specifies the font for an element.

font-size

The font-size CSS property sets the size of the font.
